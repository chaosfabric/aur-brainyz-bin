# brainyz-bin — AUR snapshot

Versioned mirror of the `brainyz-bin` package published to the
[Arch User Repository](https://aur.archlinux.org/packages/brainyz-bin).

**This repo is a mirror for history and review.** To install, use any
AUR helper:

```sh
yay -S brainyz-bin      # or paru, pikaur, etc.
```

Upstream project: FavitoX/brainyz.

## Files
```PKGBUILD``` — the package build recipe.
```.SRCINFO``` — generated with ```makepkg --printsrcinfo > .SRCINFO```; required by AUR alongside the PKGBUILD.
